KEYBOARD
SPACE : start and stop simulation
R : reset the simulation
SHIFT + R : reload the scene file
K : reload klean! (same as shift+R)
C : clear the scene (everything!) H : hide/display overlay
O : hide/display overlay
ENTER : start and stop recording
M : simulate up to next merge or unmerge
L : load a file
F : load as a factory
Q : Spin all "Spinner" bodies around spinning axis
W : Spin all "Spinner" bodies in opposite direction to spinning axis
ALT + Q : move controllable springs up
ALT + E : move controllable springs down
ALT + W : move controllable springs forward
ALT + S : move controllable springs backward
ALT + A : move controllable springs left
ALT + D : move controllable springs right
ALT + &uarr : decrease length of controllable spring
ALT + &darr : increase length of controllable spring
&uarr : double step size and sub steps
&darr : half step size and half sub steps (stops at 1 substep)
&larr : previous scene
&rarr : next scene
J : jiggle all bodies
. : factory run toggle
, : factory create a single body
MOUSE
left drag : rotate scene
middle drag : translate scene
right drag : dolly in and out
SHIFT + left drag : mouse spring
CTRL + middle drag : mouse impulse
CTRL + left drag : z rotation
