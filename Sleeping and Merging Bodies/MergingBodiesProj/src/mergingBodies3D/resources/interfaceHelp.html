KEYBOARD
SPACE : start and stop simulation
R : reset the simulation
C : clear the scene (everything!) ENTER : start and stop recording
M : simulate up to next merge or unmerge
L : load a file
F : load as a factory
&uarr : double step size and sub steps
&darr : half step size and half sub steps (stops at 1 substep)
&larr : previous scene
&rarr : next scene
J : jiggle all bodies . : factory run toggle , : factory create a single body MOUSE
left drag : rotate scene
middle drag : translate scene
right drag : dolly in and out
shift + left drag : mouse spring
